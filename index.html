PROJECTS:

Bulk Upload Product Images from Google Drive to Shopify

 Python (Pandas), Google Drive API, Shopify

The objective of this project was to write Python code that would allow Shopify store owners to bulk upload product images from a Google Drive folder to a Shopify store, via Shopify's "import products by CSV" feature. Image file information is collected via the Google Drive API, and is formatted using Pandas.

Bike Share Toronto Rides Analysis

 Python (Pandas, Rasterio, JSON, seaborn)

For this project, I analysed Bike Share Toronto data to find that riders tended to lose elevation (i.e. bike downhill more often than uphill) in 2022. This project required me to collect data from three different sources: ride data (CSV), station data (JSON), and elevation data (DEM/TIF). Seaborn was used to create visualizations.

Italian Bakery Online Store

 Shopify (HTML, CSS, JavaScript, Liquid)

This is an ecommerce website that I created, pro bono, for an Italian bakery that was forced to shift its business online during the pandemic, due to mandated store closures. It was completed using Shopify (including several third-party apps and custom CSS, JavaScript, and Liquid). I communicated with bakery owners in order to obtain required assets (logos, photos, copy, etc.) and understand business needs surrounding product categorization, checkout, and delivery.

U.S. National Parks Recreational Visits Analysis

 R (dplyr, ggplot2), Tableau

This is a project I started after completing the Coursera Google Data Analytics Professional Certificate. Since my goal was to practice using R for data aggregation and visualization, I chose to explore rather obvious hypothesis: more people will visit U.S. national parks when the weather is warm.
