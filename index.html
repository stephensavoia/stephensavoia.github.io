PROJECTS:

Bike Share Toronto Rebalancing Minimizer

 Python (Pandas, Rasterio, scikit-learn, plotly)

This is an ongoing machine learning project. The goal is to create a tool that can predict a proposed bike share station's burden on the rebalancing team (i.e. the employees that use trucks to move bikes from one station to another), using the proposed station's longitude, latitude, and elevation.

Bike Share Toronto Rides Analysis

 Python (Pandas, Rasterio, JSON, seaborn)

For this project, I analysed Bike Share Toronto data to find that riders tended to lose elevation (i.e. bike downhill more often than uphill) in 2022. This project required me to collect data from three different sources: ride data (CSV), station data (JSON), and elevation data (DEM/TIF). Seaborn was used to create visualizations.

Italian Bakery Online Sales Analysis

 Python (Pandas, Matplotlib), Slides

The objective of this project was to create a visualization that would allow owners of an Italian bakery to see how sales of cannoli had changed (if at all) after the introduction of new packaging. In order to complete this project, I isolated specific product data using a regex mask, aggregated sales data by month, and created a dual-axis line graph.

U.S. National Parks Recreational Visits Analysis

 R (dplyr, ggplot2), Tableau

This is a project I started after completing the Coursera Google Data Analytics Professional Certificate. Since my goal was to practice using R for data aggregation and visualization, I chose to explore rather obvious hypothesis: more people will visit U.S. national parks when the weather is warm.
