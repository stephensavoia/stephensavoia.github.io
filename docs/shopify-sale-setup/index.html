The Best Way to Set Up a Sale in Shopify

Running effective sales is crucial for Shopify merchants. Promotions drive conversion and generate excitement around a brand. It's therefore unfortunate that Shopify doesn't offer a particularly strong built-in method for setting up sales. Below is a clear guide to help Shopify merchants set up sales in the most effective way possible.

The Problem With Using Compare at Prices to Set Up a Sale

The simplest way to run a sale in Shopify is to lower a variant's price and set its compare at price. The online store's theme will then display the discounted price alongside a slashed-out original price, signaling a sale to customers.

While this approach is simple and easy to implement, it creates two problems:

  1. Even though the product page and cart display the slashed-out compare-at price, the checkout page does not. At checkout, customers see the lower price with no indication that they're receiving a discount. This can create confusion and hurt conversion.
  2. Since this method involves lowering the variant's actual price, any existing discount codes or automatic discounts will stack on top of the sale price. To prevent this, merchants must either (a) manually disable all other discount codes—including those created by third-party apps—or (b) install a discount-blocking app.

The Problem With Automatic Discounts

The issues with using compare at prices to set up a sale in Shopify lead some merchants to use automatic discounts instead. This approach, however, has its own downside. While automatic discounts display both the discounted price and the slashed-out original price in the cart (on most themes) and at checkout, there is no way to show discount prices on the product or collection pages. This is terrible for conversion, because a product page that doesn't display a sale price with a slashed-out original price is far less likely to convert.

The Best Way to Set Up a Sale in Shopify

The best way to set up a sale in Shopify is to use an automatic discount and then create a product variant metafield called "discount price" that can be displayed on the product page.

Merchants can accomplish this by following these steps:

  1. Create an automatic discount (you may need to create multiple discounts, if different products will be discounted by different percentages).
  2. Have a developer create a product variant metafield called "discount price."
  3. Set the value of each variant's "discount price" metafield to its sale price. If your store has a lot of products, you can use the app Matrixify to help speed this up.
  4. Have a developer update your theme so that, when the "discount price" metafield has a value, it displays that value as the product price and shows the regular price as a slashed-out, compare-at price.

That's it! After following these steps, you will have set up your sale in a way that (a) displays savings on the product page, cart, and checkout page and (b) prevents customers from stacking coupon codes on top of the sale price.

Optional: Scheduling Your Sale for a Future Date

Sometimes merchants want to schedule a sale to start at a specific time in the future—maybe overnight or on a weekend. To achieve this while using the sale implementation method described above, two additional steps are required:

  1. Schedule a start time for the automatic discount.This is easy to accomplish, as scheduled start and end times are built into Shopify's automatic discount feature.
  2. Have a developer create a shop-level boolean metafield called "show discount prices" and set it to false. Then, using Shopify Flow, schedule a workflow that updates this metafield to true at the desired time. Finally, have the developer update the theme code to conditionally display the discount price only when (1) the "discount price" metafield is not blank, and (2) the shop-level "show discount prices" metafield is set to true. With this setup, all "discount price" variant metafields can be input in advance, and the theme will reveal them to customers only when the scheduled time arrives.
